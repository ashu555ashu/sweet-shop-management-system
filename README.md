# Sweet Shop Management System

A Django-based web application for managing a sweet shop. Core functionality for accounts, orders, sweets, and cart operations is implemented.

## Features

- **User Management**: Registration, login, and staff management.
- **Shop Management**: Add, view, and manage sweets.
- **Cart Management**: Add items to cart, view cart, and place orders.
- **Order Management**: View orders, order details, and order confirmation.
- **Media Handling**: Upload images for sweets and products.
- **Django Admin**: Full access to manage models via admin panel.

## Tech Stack

- **Backend**: Django (Python)
- **Database**: SQLite
- **Frontend**: HTML, CSS, JavaScript
- **Version Control**: Git & GitHub
